# ⚡ Calculadora de Consumo de Energia

## 📌 Objetivo
Este projeto calcula o consumo mensal de energia de um aparelho, e o gasto em reais.

## 🐍 Linguagem
Python

## 🧮 Fórmula
consumo = (potência × horas por dia × 30) / 1000
gasto= consumo * 0,75

## ▶️ Como usar
1. Execute o arquivo app.py
2. Digite as informações pedidas
3. Veja o resultado na tela

## 💡 Exemplo
Aparelho: Geladeira  
Consumo: 45 kWh/mês
